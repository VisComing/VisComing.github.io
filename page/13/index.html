安装pyenv

发表于 2021-06-07 阅读次数： Valine：

1、首先安装构建python需要的工具官网教程

对于Ubuntu，执行下列命令（注意，命令可能会过时，以官网为准）

2、clone pyenv的代码库，clone到~/.pyenv

git clone https://github.com/pyenv/pyenv.git ~/.pyenv

如果无法访问github，那么可以使用镜像

git clone https://github.com.cnpmjs.org/pyenv/pyenv.git ~/.pyenv

3、将pyenv添加到环境变量

刚才克隆的pyenv代码库其中已经包含了编译好的pyenv文件了，在~/.pyenv/bin/中，我们只需要将其加入环境变量中就可以了。

4、此时已经安装好了pyenv，在命令行中输入pyenv应该可以看见以下输出

5、下载python

如果直接用pyenv的命令安装python，它是从python官网下载python包的，比较慢。我们可以先从镜像下载好放到~/.pyenv/cache中，然后再执行pyenv install 3.x.x，pyenv会先在cache中查找相应的安装包，如果找到了就直接安装。

v=3.7.3;wget https://npm.taobao.org/mirrors/python/$v/Python-$v.tar.xz -P ~/.pyenv/cache/;pyenv install $v

v表示python版本号，然后从淘宝源上下载放进~/.pyenv/cache中，然后安装。

6、安装好后，执行pyenv global 3.x.x就可以让安装好的python全局生效了。
